I have imported data using read.csv to load marketing_data. Functions like nrow() and ncol() are used to get the number of rows and columns in the dataset.Glimpse() provides a quick overview of the data, showing the structure, types, and first few entries of each column. It is useful for quickly understanding the data's composition and spotting any anomalies.

Summary() provides a statistical summary of the numerical data, including metrics like the mean, median, minimum, maximum, and quartiles. This helps identify patterns, trends, or outliers in the data.

The code checks for missing values using is.na().The which() function finds the positions of missing values, while sum(is.na()) calculates the total number of missing values. This is crucial for data cleaning and ensuring the dataset is suitable for analysis.

The table(duplicated()) function checks if there are any duplicate rows in the dataset. Identifying and handling duplicates is important for maintaining data quality and ensuring accurate analysis.

Code

marketing_data <- read.csv("C:/Users/kylew/OneDrive/Desktop/uni/MMA marketing_data_sample.csv")

nrow(marketing_data)
ncol(marketing_data)
glimpse(marketing_data)
numerical_data <- select(marketing_data, where(is.numeric))
summary(numerical_data)

### Handing missing data

which(is.na(marketing_data))
sum(is.na(marketing_data))

table(duplicated(marketing_data))
